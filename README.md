## Pull requset 방법
1. 해당 프로젝트의 저장소를 본인 계정의 저장소로 Fork 합니다.
2. 로컬 작업을 위해서 Fork한 저장소를 Clone 합니다.
3. 로컬에서 작업 후 commit, push 하여 작업 내용을 저장소에 업로드합니다.
4. pull request를 통해 작업내용을 반영할 수 있도록 요청합니다.
